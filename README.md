
# TODO App

This TODO application based on django web framework

Django v4.0.6
## Installation

Run app

```bash
  git clone https://github.com/slicetopieces/todoapp-django.git
  cd todoapp-django
  pip install -r requirements.txt
  python manage.py runserver
```
```
  Default superuser credentials
  admin:admin
```
