
# TODO App

CRUD application based on Django web framework

Django v4.0.6
## Installation

```bash
  git clone https://github.com/slicetopieces/todoapp-django.git
  cd todoapp-django
  pip install -r requirements.txt
  python manage.py runserver
```
Credentials
```
  admin:admin
```
